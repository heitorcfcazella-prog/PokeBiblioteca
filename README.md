# PokeBiblioteca

## Descrição
Aplicação web que busca e apresenta informações de pokémons utilizando APIs públicas.

## Funcionamento: 
1. O usuário digita o nome de um pokémon;
2. Clica em "Mostrar Card";
3. As informações do Pokémon (nome, vida, tipo(s), altura e peso) aparecem em um card abaixo;
4. O usuário pode solicitar até 9 cards, caso ultrapasse esse limite o último card é deletado;
5. O usuário pode fechar os cards individualmente ou todos de uma vez;

## Tecnologias:
- HTML
- CSS
- JavaScript Vanilla
- PokédexAPI (Para dados gerais)
- PokéAPI (Para altura e peso);
